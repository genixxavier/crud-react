# Crud react - Localstorage
Se uso:
localstorage para la data
Sass como procesador de css
react-router-dom
## Demo
https://crud-react-gnxcode.netlify.app/
## Instalar en localhost

Dependencies:
node v12.18.1
```sh
npm install
```

Para producción:

```sh
npm run build
```

inicar proyecto en mode desarrollo:

```sh
npm start ó npm run start
```
